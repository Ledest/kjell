What is Kjell?

Kjell (IPA: [/ʃɛl/]) is a refurbishing of the Erlang Shell, with some eye-candy and plugin support added.

Build

To build Kjell, execute the following commands in a terminal window :


$ git clone --recursive https://github.com/karlll/kjell.git
$ cd kjell
$ make

Run

To start kjell, run the following in the same directory as the above.

bin/kjell

Dependencies

To build Kjell the following is needed

  • Autoconf
  • Curses
  • gcc
  • Erlang/OTP R16B01+
  • A power-line patched font (for using extension kjell-prompt)
As of now, Kjell only supports Unix and MacOs

Version information

  • 0.2.0 - Extension support, no custom OTP build needed
  • 0.1.0 - Initial version
