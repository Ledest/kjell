kjell
=====

A Refurbished Erlang Shell

![Kjell](http://karlll.github.io/kjell/images/kjell_demo_1.png)

See http://karlll.github.com/kjell/

## Dependencies

* Erlang/OTP R16B01+

## Build
~~~
  $ git clone https://github.com/karlll/kjell.git
  $ make
~~~  
## Run
~~~
  $ bin/kjell
~~~
